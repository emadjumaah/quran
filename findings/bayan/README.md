# باب البيان — جلسة التصميم (2026-07-17)

جلسة بحث وتصميم مستقلة (فرع `bayan-design`) — لا بناء على التطبيق ولا أسراب.
التكليف الكامل في `findings/BAYAN-SESSION-PROMPT.md`.

## المحتويات

- **DESIGN.md** — تصميم الباب: الطبقات الثلاث، نموذج البيانات، حدود ما
  يُدَّعى، تصور الواجهة.
- **SOURCES.md** — جرد المصادر: المحلي الجاهز، والرقمي العام (OpenITI بعد
  انحجاب الشاملة عن الجلب الآلي)، والمحفوظ الحقوق، وترتيب الأولوية.
- **PROPOSAL.md** — العرض على المالك: المراحل والأحجام والبوابات والقرارات
  الستة المطلوبة.
- **cards/** — خمس بطاقات تجريبية مكتملة يدويًّا (إثبات الجدوى):
  1. أتى / جاء
  2. خوف / خشية (مع طبقة تحشية يدوية موسومة)
  3. اسطاعوا / استطاعوا (الكهف ٩٧)
  4. القاسطون / المقسطون
  5. بصائر
- **queries/usage_map.py** — سكربت خريطة الاستعمال الحتمي (مدخله مواصفة
  البطاقة، مخرجه خريطة كاملة موضعًا موضعًا).
- **maps/** — الخرائط المولدة للبطاقات الخمس (ناتج السكربت كما هو).
